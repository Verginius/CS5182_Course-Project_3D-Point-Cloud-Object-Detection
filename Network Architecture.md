# SECOND Network Architecture

## 概述

SECOND (Sparsely Embedded Convolutional Detection) 是一种用于自动驾驶 3D 点云检测的模型，基于稀疏卷积实现高效的特征提取和检测。

## 符号说明

| 符号 | 含义 | 典型值 |
|------|------|--------|
| **B** | Batch Size (批量大小) | 1-4 (训练/推理) |
| **N** | Number of Voxels (体素数量) | ~80,000 (KITTI典型) |
| **D** | Depth (深度)，Z轴体素数 | 40 |
| **H** | Height (高度)，Y轴体素数 | 4000 |
| **W** | Width (宽度)，X轴体素数 | 7504 |
| **C** | Channels (通道数)，特征维度 | 变 |

## 网络配置

| 配置项 | 值 |
|--------|-----|
| Grid Size | `[40, 4000, 7504]` (Z, Y, X) |
| Point Cloud Range | `[0, -40, -3, 150.08, 40, 1]` |
| Voxel Size | `[0.02, 0.02, 0.1]` |
| Downsampling | 8x |
| Output Feature Map | `250 × 469` (Y/8 × X/8, Z轴压缩后) |
| 体素分辨率 | `0.02m` |
| 感知范围 | X: [0, 150m], Y: [-40, 40m], Z: [-3, 1m] |
| Batch Size | B = 1-4 |
| 典型体素数 | N ≈ 80,000 |

---

## 1. MeanVFE (Voxel Feature Encoder)

对体素内所有点特征取平均，压缩点数维度。

| 层 | 输入 Shape | 输出 Shape | 说明 |
|----|-----------|-----------|------|
| MeanVFE | `[N, MaxPoints, 4]` | `[N, 4]` | N≈80,000, MaxPoints=32 |

---

## 2. VoxelBackbone8x (稀疏3D主干网络)

**注**: 稀疏卷积仅处理非空体素，N随网络深度递减

### 2.1 Conv1 (Stage 1 - 保持分辨率)

| 层 | 输入 Shape | 输出通道 | 输出 Shape | 类型 |
|----|-----------|---------|-----------|------|
| SubMConv3d | `[N, 40, 4000, 7504, 4]` | 16 | `[N, 40, 4000, 7504, 16]` | SubM |

### 2.2 Conv2 (Stage 2 - 2x下采样)

| 层 | 输入 Shape | 输出通道 | 输出 Shape | 类型 |
|----|-----------|---------|-----------|------|
| SpConv3d (stride=2) | `[N₁, 40, 4000, 3752, 16]` | 32 | `[N₂, 20, 2000, 3752, 32]` | SpConv |
| SubMConv3d | `[N₂, 20, 2000, 3752, 32]` | 32 | `[N₂, 20, 2000, 3752, 32]` | SubM |
| SubMConv3d | `[N₂, 20, 2000, 3752, 32]` | 32 | `[N₂, 20, 2000, 3752, 32]` | SubM |

> N₁≈80,000 (输入), N₂≈N₁/8≈10,000 (稀疏化后)

### 2.3 Conv3 (Stage 3 - 2x下采样)

| 层 | 输入 Shape | 输出通道 | 输出 Shape | 类型 |
|----|-----------|---------|-----------|------|
| SpConv3d (stride=2) | `[N₂, 20, 2000, 3752, 32]` | 64 | `[N₃, 10, 1000, 1876, 64]` | SpConv |
| SubMConv3d | `[N₃, 10, 1000, 1876, 64]` | 64 | `[N₃, 10, 1000, 1876, 64]` | SubM |
| SubMConv3d | `[N₃, 10, 1000, 1876, 64]` | 64 | `[N₃, 10, 1000, 1876, 64]` | SubM |

> N₃≈N₂/8≈1250 (进一步稀疏化)

### 2.4 Conv4 (Stage 4 - 2x下采样 + Z轴压缩)

| 层 | 输入 Shape | 输出通道 | 输出 Shape | 类型 |
|----|-----------|---------|-----------|------|
| SpConv3d (stride=2) | `[N₃, 10, 1000, 1876, 64]` | 128 | `[N₄, 5, 500, 938, 128]` | SpConv (压缩Z轴) |
| SubMConv3d | `[N₄, 5, 500, 938, 128]` | 128 | `[N₄, 5, 500, 938, 128]` | SubM |
| SubMConv3d | `[N₄, 5, 500, 938, 128]` | 128 | `[N₄, 5, 500, 938, 128]` | SubM |



### 2.5 Conv5 (Stage 5 - 2x下采样)

| 层 | 输入 Shape | 输出通道 | 输出 Shape | 类型 |
|----|-----------|---------|-----------|------|
| SpConv3d (1, 2, 2) | `[N₄, 5, 500, 938, 128]` | 128 | `[N₅, 5, 250, 469, 128]` | SpConv (Z轴保持) |
| SubMConv3d | `[N₅, 5, 250, 469, 128]` | 128 | `[N₅, 5, 250, 469, 128]` | SubM |



---

## 3. Map to BEV (稀疏转稠密)

将 3D 稀疏张量转换为 2D BEV 伪图像。

| 操作 | 输入 Shape | 输出 Shape |
|------|-----------|-----------|
| `spconv.dense()` | `[B, 128, 5, 250, 469]` (稀疏) | `[B, 128, 5, 250, 469]` (稠密) |
| `view()` (flatten D) | `[B, 128, 5, 250, 469]` | `[B, 640, 250, 469]` |

> **关键**: `128 channels × 5 depth = 640 channels`
> 
> 典型值: B=1-4, 通道640, 空间250×469

---

## 4. CenterHead (Anchor-Free 检测头)

基于 CenterPoint 范式的 Anchor-Free 检测头，预测中心点热力图、3D尺寸、深度和朝向角。

### 4.1 Shared Conv Layers

| 层 | 输入 Shape | 输出通道 | 输出 Shape |
|----|-----------|---------|-----------|
| Conv2d + BN + ReLU | `[B, 640, 250, 469]` | 256 | `[B, 256, 250, 469]` |
| Conv2d + BN + ReLU | `[B, 256, 250, 469]` | 256 | `[B, 256, 250, 469]` |

### 4.2 Heatmap Head (中心点检测)

| 层 | 输入 Shape | 输出通道 | 输出 Shape |
|----|-----------|---------|-----------|
| Conv2d + BN + ReLU | `[B, 256, 250, 469]` | 128 | `[B, 128, 250, 469]` |
| Conv2d | `[B, 128, 250, 469]` | 3 | `[B, 3, 250, 469]` |

> **输出**: 每类一个 heatmap channel

### 4.3 Regression Head (框回归)

| 层 | 输入 Shape | 输出通道 | 输出 Shape |
|----|-----------|---------|-----------|
| Conv2d + BN + ReLU | `[B, 256, 250, 469]` | 128 | `[B, 128, 250, 469]` |
| Conv2d | `[B, 128, 250, 469]` | 7 | `[B, 7, 250, 469]` |

> **输出**: 7个回归参数 (w, l, h, sin, cos, z, depth)

---

## 输出矩阵汇总

| 输出 | Shape | 典型值 (B=4) | 说明 |
|------|-------|--------------|------|
| **heatmap** | `[B, 3, 250, 469]` | `[2, 3, 250, 469]` | 3类中心点热力图 |
| **regression** | `[B, 7, 250, 469]` | `[2, 7, 250, 469]` | w, l, h, sin, cos, z, depth |


## 回归参数详解

每个类别的 7 个回归参数：

| 索引 | 参数 | 说明 |
|------|------|------|
| 0 | w | 宽度 (log尺度) |
| 1 | l | 长度 (log尺度) |
| 2 | h | 高度 (log尺度) |
| 3 | sin(θ) | 航向角正弦值 |
| 4 | cos(θ) | 航向角余弦值 |
| 5 | z | 高度偏移 |
| 6 | depth | 深度 |

## 计算流程图

```
Input Point Cloud (N points)
         │
         ▼
┌─────────────────┐
│   MeanVFE       │  [N, MaxPoints, 4] → [N, 4]
└─────────────────┘
         │
         ▼
┌─────────────────┐
│ VoxelBackbone8x │  Sparse 3D CNN
│  conv1: 4→16    │  40×4000×7504
│  conv2: 16→32   │  20×2000×3752
│  conv3: 32→64   │  10×1000×1876
│  conv4: 64→128  │  5×500×938
│  conv5: 128→128 │  5×250×469
└─────────────────┘
         │
         ▼
┌─────────────────┐
│   Map to BEV    │  dense() → view()
│                  │  [B,128,5,250,469] → [B,640,250,469]
└─────────────────┘
         │
         ▼
┌─────────────────┐
│   CenterHead    │  Anchor-Free 2D CNN
│  shared: 640→256│
│  heatmap: →3    │  中心点热力图
│  regression: →7 │  框参数
└─────────────────┘
         │
    ┌────┴────┐
    ▼         ▼
heatmap   regression
[B,3,250,469] [B,7,250,469]
```
